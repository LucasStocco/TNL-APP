# 📱 TNL - Lista de Compras

Projeto de lista de compras com backend em Spring Boot e frontend em Flutter.

---

## 📌 Resumo da funcionalidade

Sistema de listas com:
- CRUD de listas
- progresso automático
- itens comprados
- resumo otimizado via backend
